
# Vacation Selector Page

#### Code Review Assignment for Ryan King, 01.15.2016

#### By Ryan King

## Description

Code review submission for Epicodus Week 2. The page takes the answers to 5 questions and outputs the best vacation spot based upon the users choices. 

## Setup/Installation Requirements

1. Open web browser of choice
2. Go to https://github.com/ryanking1/vacationPage
3. Clone the repository so you have all the files locally
4. Open vacationPage.html in your web browser of choice
5. Complete the form and get your answer


## Known Bugs
Currently only has placeholder binary values that need to be replaced with actual vacation spots. However, I'm not knowledgable enough on world travel to complete an accurate list without a bit of research.
